# Twitter Clone Frontend

Frontend du clone de Twitter développé avec **React**.

Cette application permet de :

* voir les tweets
* poster des tweets
* liker / retweeter
* suivre des utilisateurs
* voir les notifications
* modifier son profil

Elle communique avec l'API **Spring Boot**.


# Installer les dépendances

```bash
npm install
```

# Lancer le projet

```bash
npm run dev
```

L'application sera disponible sur :

```
http://localhost:5173
```
